About Me
Welcome to my website. My name is David Sung and I'm an aspiring Web Developer. I started this page as a personal project to hone my Web Developing skills, something I got interested way back in middle-school toying around with MySpace layouts.

 This website was initially created from scratch using entirely HTML/CSS and custom JavaScript/jQuery functions to animate elements on the page. To familiarize myself with different JavaScript Frameworks and expand my knowledge of front-end design, I decided to integrate the BootStrap framework into the website to clean up a lot of the code.
